## 第一个python程序

* Pycharm创建项目
* 设置项目编码UTF-8   
  打开设置+ file enconding 全部设置为utf-8
* 编写第一个程序   
`print("hello world") `  右键运行,控制台输出hello world, 第一个python程序完成

* print内置函数的更多用法
```
print('hello world')
#  多个字符串,默认空格拼接
print('start study python','hello world')
# 输入,然后等待输出
print('please input a world:')
name= input();
print(name)
```
