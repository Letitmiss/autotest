# 逻辑控制

## 条件判断
* 语法
```
if <条件判断1>:
    <执行1>
elif <条件判断2>:
    <执行2>
elif <条件判断3>:
    <执行3>
else:
    <执行4>
```
* 脚本实例
```
# if 判断 else
age = 3
if age > 18:
    print("age is > 18")
else:
    print("age is <=18")

# elif
age = 3
if age > 18:
    print("age is > 18")
elif age > 2:
    print("age is <=18 and > 2")
else:
    print("age is <= 2")
# 从上往下判断, 如果一个是true,后面的判断忽略执行

# 只要x是非零数值、非空字符串、非空list等，就判断为True，否则为False
x = 0
if x:
 print('True')
else:
  print('False')

x = 1
if x:
  print('True')
else:
  print('False')

x = ''
if x:
  print('True')
else:
  print('False')  #false

x = []
if x:
  print('True')
else:
  print('False')  # false

x = ['123']
if x:
  print('True')
else:
  print('False')

```
